  • npm install -g typescript -> cài đặt type script (nếu muốn biết máy đã cài đặt? gõ lệnh -> tsc -v)
  • tsc --init -> tải về tsconfig.json để sửa lại bộ biên dịch
    • "target": "es6",
    • "module": "commonjs",
    • "watch":true, -> tự động dịch khi save
    • "outDir": "./js", -> thư mục chứa .js đầu ra
    • "rootDir": "./ts", -> thư mục chứa .ts cần dịch
    • "removeComments": true, -> bỏ comment
    • "strict": true,
    • "moduleResolution": "node",
    • "esModuleInterop": true,
    • đặt dòng json thứ hai "exclude": ["css", "js"] -> loại bỏ các thư mục không biên dịch

  • Xử lý Ajax bằng vanilla js
  • var xhr = new XMLHttpRequest(); -> tạo req
    • lấy data json gửi đến
    • xhr.open('GET','tên file cần lấy data or tệp json',true); -> nếu là file khác gửi tới chọn tùy theo phương thức gửi
    • xhr.onload = function () {if (this.status == 200) {}} -> kiểm tra mã 200 ở server success
    • var data = JSON.parse(this.responseText); -> gán data bằng json lấy từ tệp nhận
    • dùng vòng lặp for(data) gán cho .innerHTML đưa vào dữ liệu
    • xhr.send();
    • gửi data đến file xử lý
    • xhr.open('POST', 'tên file', true);
    • xhr.setRequestHeader('Content-type', 'application/x-www-form-urlencoded');
    • xhr.onload = function () {if (this.status == 200) {console.log(this.responseText)}} -> kiểm tra data gửi đi
    • xhr.send(params); -> params là dữ liệu được lấy từ client (params = 'title='+data;)
